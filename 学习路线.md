# NLP 聊天机器人实习 · 学习路线

> 目标：**深刻掌握** 词向量 → RNN/LSTM → Attention → Transformer
> 原则：**先建直觉（看图解/视频）→ 再读论文 → 再亲手写代码撞瓶颈**。不跳过"撞瓶颈"这一步，它是理解下一个技术的动机。
> 配套：本路线与实战代码一一对应（[data_pipeline.py](data_pipeline.py)、[seq2seq.py](seq2seq.py)、后续 attention/transformer）。

---

## 全局：这条技术链在解决什么

```
词向量(Word2Vec)   把字/词变成向量，机器才能计算
      ↓
RNN / LSTM         处理变长序列、有"记忆"
      ↓
Seq2Seq            两个LSTM：读问句→压成一个固定向量→生成答句
      ↓  ❗瓶颈：整句压成一个向量，长句信息丢失
Attention          生成每个字时回头"看"问句不同部分 → 解决瓶颈
      ↓  ❗仍依赖RNN：不能并行、长距离依赖弱
Transformer        扔掉RNN，"只用注意力" → 现代大模型基石
      ↓
文本生成 / 聊天机器人
```

每一步都是为了解决上一步的缺陷。理解了"动机"，就理解了整条链。
